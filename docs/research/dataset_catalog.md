# Available Dataset Catalog

## Overview

ECHOLab maintains a number of shared datasets and data products that are used across multiple projects. This page serves as a directory of commonly used data resources and where they can be found.

The goal of this page is to help researchers discover existing data resources before creating new copies or repeating data acquisition and processing work.

This page is intended to provide a high-level inventory rather than detailed documentation. Individual datasets may have separate documentation pages describing acquisition, processing, versioning, and usage considerations.

For information on data governance, storage policies, and working with Oak, see the related documentation pages.

> **Note:** Location refers to the current location of the dataset and not necessarily its long-term canonical location. As ECHOLab continues migrating shared resources to Oak, dataset locations may change.
