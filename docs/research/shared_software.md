# Shared Software and Code Resources

This page catalogs software, R packages, and reusable code resources developed by ECHOlab researchers that may be useful across projects.

Before building a new workflow from scratch, it is worth checking whether similar tools already exist.

## Heat

Repository: https://github.com/echolab-stanford/heat

Tools for processing gridded environmental datasets and generating exposure metrics for geographic units. The package was developed to support environmental epidemiology and climate-health analyses, but many functions are broadly applicable to spatial exposure assessment.

Potential applications:

* Temperature exposures
* Air pollution exposures
* Population-weighted aggregation
* Nonlinear exposure transformations
* Aggregation of gridded datasets to polygons

## DHS.tools

Repository: https://github.com/echolab-stanford/DHS.tools

Tools for working with Demographic and Health Surveys (DHS) data. The package includes utilities for processing survey data and constructing commonly used measures.

Potential applications:

* DHS data processing
* Wealth index construction
* Survey harmonization
* Reproducible DHS workflows

## Contributing

If you have developed software, workflows, or reusable code that may be useful across projects, consider adding it to this page.

The goal is not to catalog every script, but to make broadly useful tools discoverable to future collaborators.
